# SmartCloud v2 — Adaptive Predictive Auto-Scaling

## 5 Novel Innovations
1. **Adaptive Ensemble Forecaster** — LSTM + ARIMA + Holt-Winters with live RMSE-based weight rebalancing
2. **SLA-Cost-Carbon Tradeoff Engine** — Multi-objective optimisation with user-adjustable weights
3. **FFT Workload Fingerprinting** — Pattern memory via cosine similarity matching
4. **XAI Anomaly Detection** — Root-cause tagging with plain-English explanations
5. **Carbon-Aware Scaling** — Green grid signal integrated into scaling decisions

## Quick Start (All Platforms)

### Windows
Double-click `start.bat`

### macOS / Linux
```bash
bash start.sh
```

### URLs
| Service    | URL                        |
|------------|----------------------------|
| Dashboard  | http://localhost:3000      |
| API Docs   | http://localhost:8000/docs |
| Prometheus | http://localhost:9090      |

### Stop
Windows: `stop.bat`  
macOS/Linux: `docker-compose down`
